- 타이머 페이지
1) 시작 버튼을 누르면 시간이 흐름
2) 중지 버튼을 누르면 시간이 멈춤
3) 구간기록 버튼을 누르면 누른 당시의 시간 정보를 저장해놨다가 표시해줌
4) 초기화 버튼을 누르면 모든 시간 정보가 사라지고 시간도 초기화(00:00:00)
    => 사이클
    초기 : (구간기록) (시작)
    시작 : (구간기록) (중지)
    중지 :  (초기화)  (계속)
